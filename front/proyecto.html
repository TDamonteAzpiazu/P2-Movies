Mi primer proyecto full-stack

Durante mi proyecto full stack, tuve la tarea de crear una página web dedicada al mundo del cine. Partiendo de mis conocimientos iniciales en HTML, CSS y JavaScript, no iba a ser suficiente, asique tocó aprender herramientas como Bootstrap para desarrollar una interfaz atractiva y responsive. También utilicé Axios para realizar solicitudes HTTP y gestionar la comunicación con MongoDB, la base de datos. En cuanto esta, aprendimos a almacenar y recuperar información sobre películas, incluyendo título, año, director, duración, géneros, puntaje y URL del póster. La página consta de dos secciones estáticas, una destinada a proporcionar información sobre el proyecto y otra con detalles de contacto. La funcionalidad principal es un formulario donde los usuarios pueden ingresar datos de películas, que luego son almacenados en la base de datos y presentados como tarjetas en la página principal. Este proyecto no solo me permitió expandir el conocimiento técnico, sino también a colaborar con compañeros para hacernos consultas, aprender a partir del código del otro y debuggear juntos.
